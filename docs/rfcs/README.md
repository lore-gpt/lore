## Proposing an RFC

1. Copy an existing RFC as a template, number it `NNNN-short-title.md`.
2. Fill in *Summary · Motivation · Design · Alternatives · Open questions*.
3. Open a PR and a Discussion thread. Small, focused RFCs get reviewed faster.
